# Edge-AI Runway FOD Detection System

An Edge-AI based runway Foreign Object Debris (FOD) and surface anomaly detection system using YOLOv10, Raspberry Pi, MQTT, and Streamlit.

## Features
- Real-time FOD detection
- Surface anomaly detection
- Streamlit dashboard visualization
- MQTT communication
- Edge AI inference using YOLO

## Technologies Used
- Python
- YOLOv10
- OpenCV
- Streamlit
- MQTT
- Raspberry Pi

## Project Structure
- dashboard.py → Streamlit dashboard
- edge_logic.py → Detection logic
- stream.py → Video streaming
- mock_pi_node.py → Raspberry Pi simulation
